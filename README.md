# Registration App

## Description

A simple registration application built using Node.js, Express, and MySQL. This application allows users to create, read, update, and delete registration entries.

## Prerequisites

- Node.js (>= 14.x)
- MySQL
- A GitHub account

### 1. Clone the Repository
